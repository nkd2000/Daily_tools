Compress

Compress small files directly in your browser. This page provides a fast, privacy-first way to reduce file sizes on-device.

Simple guide

  1. Choose a file.
  2. Select compression options (lossy / lossless).
  3. Preview and download the compressed file — nothing leaves your browser.

If you want additional features (batch compress, server-side options), open the Contact page and request them.
